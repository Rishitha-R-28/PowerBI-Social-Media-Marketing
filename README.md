# PowerBI-Social-Media-Marketing
Social Media Marketing Analysis using Power BI
## Project Overview
This project focuses on analyzing social media marketing performance using Power BI to help understand how different platforms and campaigns perform over time. The dashboard transforms raw marketing data into interactive visual insights that support data-driven marketing decisions.

The analysis highlights key performance metrics such as engagement, reach, impressions, and campaign effectiveness, enabling marketers to identify trends, strengths, and improvement areas.
## Tools Used
Power BI – Dashboard creation and visualization

Power Query – Data cleaning, transformation, and modeling

DAX (Data Analysis Expressions) – Custom measures and KPIs

Excel / CSV Dataset – Source data for analysis.
## Key Insights & Findings
Identified top-performing social media platforms based on engagement and reach

Compared campaign performance to understand which strategies deliver better results

Observed engagement trends over time to support future marketing planning

Helped translate raw marketing data into actionable business insights.
